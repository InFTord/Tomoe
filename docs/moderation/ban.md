# Ban
## Summary
Bans a member from the guild.

## Description
When banning, Tomoe will attempt to send the victim a private message before banning, as such:
![ban_overload_1.png](/docs/images/ban_overload_1.png)

## Overloads
- `ban <user ids>`
- `ban <user mentions>`
- `ban <users> [banReason]`
- `ban <users> [pruneDays = 7] [banReason]`

>>ping
